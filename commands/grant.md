# blue grant

Find and prepare grant applications for Base ecosystem funding.

## What it does

Maps your project to available grants and generates a scored application draft.

## Grant programs covered

- Base Ecosystem Fund (Coinbase)
- Optimism RetroPGF
- Ethereum Foundation grants
- Gitcoin rounds (Base-focused)
- Bankr ecosystem grants
- Protocol-specific grants (Uniswap, Aave, etc.)

## Required input

- Project name
- What it does (one paragraph)
- Current stage (idea / prototype / live)

## Optional input

- Team background
- Requested amount
- GitHub URL
- Website

## Output

- Grant match list (best fits ranked by score)
- Application draft for top match
  - executive summary
  - problem and solution
  - traction bullets
  - milestones with deliverables
  - budget breakdown
  - team credibility points
- Scoring vs Base/Coinbase criteria (innovation, feasibility, impact, team)
- Suggested grant size and ask framing
- Weak points to address before submitting

## Usage

```
blue grant
```
