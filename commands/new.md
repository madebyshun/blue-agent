# blue new

Guided full workflow: Idea → Build → Audit → Ship in one command.

## What it does

Runs the complete founder pipeline end-to-end without switching between commands.

## Flow

1. Collect your idea in one sentence
2. Generate fundable brief (`blue idea`)
3. Generate build plan + stack (`blue build`)
4. Run risk and security review (`blue audit`)
5. Output deploy checklist (`blue ship`)

## Output

- Idea brief (problem, why now, why Base, GTM, risks, 24h plan)
- Build plan (architecture, stack, files, steps)
- Audit report (risks, blockers, go/no-go)
- Deploy checklist (steps, verification, release notes)

## Usage

```
blue new
```

Then answer the prompts. Everything else is automated.
