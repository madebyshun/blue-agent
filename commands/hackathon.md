# blue hackathon

Optimize your project for a Base hackathon submission.

## What it does

Tailors your idea and build plan for hackathon judging criteria.
Outputs a submission-ready package in hours, not days.

## Supported events

- ETHGlobal (Bangkok, Singapore, Brussels, NYC…)
- Based Latam
- Onchain Summer Buildathon
- Coinbase / Base ecosystem hackathons
- Any custom hackathon (provide theme + criteria)

## Required input

- Your idea (one sentence)
- Hackathon name and theme

## Optional input

- Team size
- Time remaining
- Tech constraints (solo, no backend, etc.)

## Output

- Focused pitch (30-second hook)
- Scoped build plan (MVP only, cut everything else)
- Demo script (what to show and in what order)
- Judge criteria mapping (how your project scores on each criterion)
- Submission checklist (README, video, deploy link, GitHub)
- Anti-scope-creep warnings

## Usage

```
blue hackathon
```
