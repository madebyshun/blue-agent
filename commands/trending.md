# blue trending

Show what's trending across builders, agents, projects, and tokens on Base.

## Usage

```
blue trending
blue trending builders
blue trending agents
blue trending tokens
```

## Required output

- Rank (1–8)
- Name and type
- Handle or identifier
- Signal (why it's trending)
- Momentum label: rising / hot / new

## Notes

- Results are LLM-grounded — based on Bankr's knowledge of recent Base ecosystem activity
- Filter by type: builders, agents, tokens, projects
- Follow up with: `blue score @handle` or `blue history @handle`
