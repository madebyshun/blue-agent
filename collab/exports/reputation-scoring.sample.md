# reputation-scoring — sample export

## What it covers

How Blue Agent should evaluate builder and agent reputation signals.

## When to use it

Use this skill when scoring outputs, ranking contributors, or comparing quality over time.

## Core concepts

- Score dimensions should be explicit.
- Weights should be stable.
- Anti-gaming matters.
- Reputation should improve with real evidence.

## Patterns

- Separate activity from quality.
- Favor repeated positive evidence.
- Penalize obvious manipulation.

## Pitfalls

- Overfitting to a single metric.
- Inflated scores without evidence.
- No explanation for score changes.

## Checklist

- [ ] Dimensions defined
- [ ] Weights defined
- [ ] Anti-gaming rules defined
- [ ] Score explanation included
