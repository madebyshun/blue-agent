# wallet-guardrails — sample export

## What it covers

Operational guardrails around wallet usage, spending, approvals, and risk controls.

## When to use it

Use this skill when wallet behavior needs safe defaults and operational boundaries.

## Core concepts

- Spend caps
- Approval thresholds
- Risk flags
- Allowlisted destinations

## Patterns

- Set conservative defaults.
- Escalate on unusual spend patterns.
- Separate low-risk from high-risk actions.

## Pitfalls

- Over-automation.
- Missing spend ceilings.
- No destination allowlist.

## Checklist

- [ ] Spend cap set
- [ ] Approval gate set
- [ ] Risk flags set
- [ ] Allowlist set
