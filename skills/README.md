# Blue Agent Skills

This folder contains grounded knowledge files that help Blue Agent reason about Base-native workflows.

## Purpose

Skills are practical Markdown references used to improve outputs for:

- `blue idea`
- `blue build`
- `blue audit`
- `blue ship`
- `blue raise`

## What a good skill looks like

- Base-specific
- Practical, not promotional
- Short enough to scan quickly
- Dense enough to be useful
- Includes risks, pitfalls, and examples

## Suggested structure

- What it covers
- When to use it
- Core concepts
- Patterns / examples
- Common mistakes
- Checklist

## Good contribution ideas

- add a new skill for a Base-native pattern
- improve examples in an existing skill
- add safety notes or guardrails
- add a new ecosystem guide
- add a new integration playbook
