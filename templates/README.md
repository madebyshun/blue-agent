# Blue Agent Templates

This folder contains starter projects and scaffolds for common Base-native use cases.

## Purpose

Templates are meant to be cloned, adapted, and shipped quickly.

## Good template traits

- opinionated but flexible
- ready to install and understand
- Base-native by default
- production-leaning
- easy to extend

## Suggested template contents

- `README.md`
- `package.json`
- `env.example`
- `scripts/` if relevant
- `contracts/` if relevant
- `tests/` or `test/`
- `blue-template.json`
- CI workflow if production-oriented
