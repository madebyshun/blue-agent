# Blue Agent — Founder Console

> AI-native founder console for Base builders 🔵

**blueagent.dev** — the web interface for Blue Agent CLI.

## CLI

```bash
npm install -g @blueagent/cli

blue idea "your idea"   # direct command
blueagent               # interactive TUI
```

## Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
