Blue Agent's x402 API lets any agent or developer pay per call in USDC on Base — no subscriptions, no API key registration required.

## How x402 works

1. Agent makes a request to a tool endpoint
2. Server returns `402 Payment Required` with payment details
3. Agent signs a USDC micropayment on Base
4. Request is fulfilled with payment proof in header

## Base URL

```
https://blueagent.dev/api/x402/
```

## Example

```bash
# First call — get payment details
curl https://blueagent.dev/api/x402/risk-gate \
  -d '{"contract": "0x..."}'

# Returns 402 with payment info
# Sign payment, retry with X-Payment header
curl https://blueagent.dev/api/x402/risk-gate \
  -H "X-Payment: ..." \
  -d '{"contract": "0x..."}'
```

<Note>
  All payments are USDC on Base (chain ID 8453). Minimum payment: $0.05.
</Note>
