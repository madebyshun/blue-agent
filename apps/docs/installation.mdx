## Packages

| Package | Command | Description |
|---------|---------|-------------|
| `@blueagent/builder` | `blue` | Core CLI — all 31 commands |
| `@blueagent/cli` | `blueagent` | Interactive TUI shell |

## Requirements

- Node.js 18+
- npm or yarn
- `BANKR_API_KEY` environment variable

## Install options

<Tabs>
  <Tab title="One-liner">
    ```bash
    curl -fsSL https://blueagent.dev/setup.sh | bash
    ```
    Installs both `@blueagent/builder` and `@blueagent/cli` globally.
  </Tab>
  <Tab title="CLI only">
    ```bash
    npm install -g @blueagent/builder
    ```
    Gives you the `blue` command.
  </Tab>
  <Tab title="TUI + CLI">
    ```bash
    npm install -g @blueagent/cli @blueagent/builder
    ```
    Gives you both `blueagent` (TUI) and `blue` (CLI).
  </Tab>
</Tabs>

## API key

```bash
# Add to your shell profile (~/.zshrc or ~/.bashrc)
export BANKR_API_KEY=your_key_here
```

## Verify install

```bash
blue --version
blue doctor
```

## Update

```bash
npm update -g @blueagent/builder @blueagent/cli
```
