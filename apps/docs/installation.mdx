title: Installation

---

## One package

```bash
npm install -g @blueagent/cli
```

Gives you two commands:

| Command | Description |
|---------|-------------|
| `blue` | Direct CLI — `blue idea "..."`, `blue audit "..."`, etc. |
| `blueagent` | Interactive TUI — arrow keys to navigate |

## Requirements

- Node.js 18+
- `BANKR_API_KEY` — set in `~/.blue-agent/config.toml` or as env var

## Set your API key

```toml
# ~/.blue-agent/config.toml
bankr_api_key = "your_key_here"
```

Or via environment:

```bash
export BANKR_API_KEY=your_key_here
```

## Verify install

```bash
blue doctor
```

## Update

```bash
npm update -g @blueagent/cli
```
