The 5 workflow commands form the complete founder loop on Base.

## blue idea

Turn a rough concept into a structured fundable brief.

```bash
blue idea [prompt]
blue idea "An AI agent marketplace for Base builders"
```

**Output:**
- Problem & insight
- Why now, why Base
- Target user & GTM
- MVP scope (in / out)
- Key risks & mitigations
- 24-hour action plan

**Price:** `$0.05` · **Model:** claude-sonnet-4-6

---

## blue build

Generate a complete technical build plan.

```bash
blue build [prompt]
blue build "Base-native staking protocol with x402 payments"
```

**Output:**
- Architecture overview
- Tech stack with reasoning
- Folder structure
- Key integrations (Base, Bankr, x402)
- Implementation steps
- Test plan

**Price:** `$0.50` · **Model:** claude-sonnet-4-6

## blue audit

Security and product risk review.

```bash
blue audit [prompt]
blue audit "my ERC-20 contract: [paste code]"
```

**Output:**
- Critical security issues (reentrancy, oracle, MEV, access control)
- Product/logic risks
- Base-specific risks
- Suggested fixes per issue
- Go / No-go recommendation

**Price:** `$1.00` · **Model:** claude-sonnet-4-6
