## blue tui

Launch the interactive terminal UI.

```bash
blue tui
# or
blueagent
```

Navigate all 31 commands with arrow keys. No flags needed — the TUI guides you through every option interactively.

**Requires:** `@blueagent/cli` installed globally.

```bash
npm install -g @blueagent/cli
```
