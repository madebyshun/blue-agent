## blue launch

Launch a fair-launch token or publish an agent.

```bash
blue launch
blue launch token
blue launch agent
```

**Modes:**
- `token` — Fair-launch ERC-20 on Base via Uniswap v4
- `agent` — Publish to the Bankr agent marketplace

---

## blue market

Browse and interact with the Bankr marketplace.

```bash
blue market
blue market list
blue market publish
blue market search "audit agent"
```
