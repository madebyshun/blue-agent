## What is Blue Agent?

**Blue Agent** is an AI agent layer built on Base. It interacts with users, automates tasks, and generates onchain activity.

Built by [Blocky Studio](https://x.com/blocky_agent), Blue Agent is the flagship AI agent of the Base ecosystem — not just a chatbot, but a full economic actor: it holds a wallet, executes onchain transactions, earns and distributes tokens, and powers a growing ecosystem of tools and services.
