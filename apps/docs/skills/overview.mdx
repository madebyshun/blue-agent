Skills are Markdown files installed to `~/.blue-agent/skills/`. Every `blue` command loads relevant skills automatically before calling the LLM — this is what makes Blue Agent Base-native instead of generic.

## Install skills

```bash
blue init
```

## The 6 default skills

| File | Used by | Description |
|------|---------|-------------|
| `base-security.md` | `blue audit` | 500+ security checks across 13 categories. Reentrancy, oracle, MEV, flash loans, access control, x402, Coinbase Smart Wallet. |
| `base-addresses.md` | `blue build`, `blue ship` | Verified contract addresses on Base — USDC, WETH, Uniswap v4, Aave, Compound. Never hallucinated. |
| `base-standards.md` | `blue idea`, `blue build`, `blue ship` | ERC standards, Base deployment patterns, x402 protocol spec, chain ID 8453 rules. |
| `bankr-tools.md` | `blue idea`, `blue chat` | Bankr LLM capabilities, all 31 x402 tools, pricing, integration patterns. |
| `blue-agent-identity.md` | `blue raise` | Blue Agent mission, product voice, tone rules. |
| `design-system.md` | `blue build` | Visual language, Tailwind tokens, card patterns, spacing. |

## Skill location

```
~/.blue-agent/
└── skills/
    ├── base-security.md
    ├── base-addresses.md
    ├── base-standards.md
    ├── bankr-tools.md
    ├── blue-agent-identity.md
    └── design-system.md
```

## Custom skills

Add any `.md` file to `~/.blue-agent/skills/` and it will be available for use. Skills are plain Markdown — no special format required.
