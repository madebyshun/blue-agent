# Blue Agent X Bot Plan

## Goal
Create the @blueagent X account as a credible build-in-public voice for Blue Agent.

## What it should do
- Post builder updates
- Share product progress
- Announce repo milestones
- Highlight Base-native tooling
- Avoid spammy growth tactics
- Stay grounded and useful

## Required files
- `docs/twitter-personality.md`
- `docs/twitter-storyline.md`

## Operating rules
- Load personality + storyline before drafting any post
- Keep a narrative log after every public post
- Route risky content to approval
- Never cold-reply
- Never mention @bankrbot without approval
- Never invent addresses, partnerships, or stats

## Good first content types
- repo progress update
- what Blue Agent is building this week
- short Base-native builder insight
- release note for a new skill/template/CLI command
- thread explaining the product direction

## Recommended launch order
1. Finish the personality + storyline files
2. Draft 5–10 safe launch tweets
3. Pick 1 launch tweet and approve manually
4. Post a few utility tweets first
5. Then enable any automation or reply sweep
