# Blue Agent Command Map

## Founder Console commands

### `blue idea`
Generates a fundable idea brief.

**Output:**
- problem statement
- why now
- why Base
- GTM outline
- risks
- first 24h plan

### `blue build`
Generates a build plan and scaffold.

**Output:**
- architecture
- recommended stack
- files to create
- implementation steps
- tests to run

### `blue audit`
Reviews the idea/build for security and product risk.

**Output:**
- risk summary
- blockers
- recommended fixes
- go/no-go decision

### `blue ship`
Prepares deployment and launch.

**Output:**
- deploy checklist
- verification steps
- release notes
- launch tasks

### `blue raise`
Creates a pitch narrative for investors or partners.

**Output:**
- summary
- market framing
- traction bullets
- ask
- investor targets

## Chat commands

### `blue chat`
Model picker + paid compute.

**Payment modes:**
- credits
- USDC
- $BLUEAGENT discount

## Launch commands

### `blue launch`
Guided agent launch wizard.

**Output:**
- agent name
- persona
- model config
- tools/skills
- pricing
- publish config

## Marketplace commands

### `blue market`
Browse or publish agents, prompts, and skills.

**Output:**
- list of assets
- filters
- creator stats
- usage counts
- purchase/use action
