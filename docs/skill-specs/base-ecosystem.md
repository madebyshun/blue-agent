# base-ecosystem

## What it covers
Base ecosystem primitives, apps, and building blocks:
- Aerodrome
- Uniswap v4
- Base Name / identity primitives
- common Base app patterns
- social and community primitives relevant to Base-native apps

## When to use
Use this skill when a task involves:
- planning a Base-native app
- choosing ecosystem integrations
- explaining Base-specific product architecture
- comparing Base primitives to generic EVM patterns

## Core guidance
- Keep everything Base-first (chain 8453)
- Prefer practical integrations over abstract ecosystem talk
- Name only verified protocols and known primitives
- Explain why a Base primitive matters for the product

## Common mistakes
- generic multi-chain advice with no Base-specific value
- treating Base like “just another EVM chain”
- inventing ecosystem partnerships or addresses

## Checklist
- [ ] Base-only scope
- [ ] ecosystem primitive identified
- [ ] integration value explained
- [ ] risks noted
- [ ] no hallucinated addresses
