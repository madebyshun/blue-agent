# base-telegram-bot

## Purpose
Starter template for a Base-native Telegram bot with wallet-aware features, grounded responses, and simple persistence.

## What it should include
- bot entrypoint
- command handlers
- basic state storage
- optional wallet integration hooks
- `README.md`
- `blue-template.json`
- `.env.example`
- CI workflow

## Default stack
- Node.js + TypeScript
- Telegram bot SDK
- JSON or lightweight storage
- Bankr LLM optional
- Base-first content and utilities

## Build goals
- launch quickly
- simple command surface
- easy to customize for communities
- safe defaults for onchain actions

## Acceptance
- bot starts cleanly
- at least one useful command works
- structure is easy to extend
- template makes Base-native setup obvious
